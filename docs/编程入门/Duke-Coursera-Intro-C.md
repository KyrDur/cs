# Introductory C Programming Specialization

## 课程简介

- 所属大学：Duke
- 先修要求：无
- 编程语言：C
- 课程难度：🌟🌟🌟🌟
- 预计学时：110 小时

非常好的课程，自我感觉收益非常大：

- 侧重基础和基本概念：如 frame、stack memory、heap memory 等讲得很透。
- 针对C最难掌握的指针，有好的练习和编程来加深和强化理解。
- 虽说课名是入门，但兼具广度和深度。
